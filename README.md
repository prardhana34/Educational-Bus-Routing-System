## 📌 Overview
The **Educational Bus Routing System** is a Java-based console application that models real-world transportation networks using graph data structures and computes the shortest path between bus stops using **Dijkstra’s Algorithm**.

This project demonstrates strong understanding of:
- Data Structures & Algorithms (DSA)
- Computer Organization & Architecture (COA)
- Software Engineering (FSE)
- Object-Oriented Programming (OOP)

---

## ⚙️ Features
- ➕ Add bus stops dynamically
- 🔗 Connect stops with weighted routes
- 📍 View all stops and routes
- 🗺️ Find shortest path between two stops
- 🚀 Efficient console-based menu system
- ✅ Smart validation and error prevention

---

## 🧠 Tech Stack
- Java (Core)
- Graph Data Structure (Adjacency List)
- HashMap & ArrayList
- PriorityQueue (Min Heap)
- Dijkstra’s Algorithm

---

## 🏗️ System Architecture
```
BusStop.java → Node representation
Graph.java → Graph structure (Adjacency List)
RouteOptimizer.java → Dijkstra Algorithm
ValidationService.java → Input validation
BusRoutingSystem.java → Controller logic
Main.java → Entry point (CLI menu)
```
---

## 📊 Algorithm Used

### 🔷 Dijkstra’s Algorithm
Used to compute the shortest path between two bus stops in a weighted graph.

### Complexity:
- Time: O((V + E) log V)
- Space: O(V + E)
