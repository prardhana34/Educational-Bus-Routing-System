# 🚍 Educational Bus Routing System

## 📌 Overview
The Educational Bus Routing System is a Java-based console application that models bus transportation routes using Graph Data Structures and computes the shortest path using Dijkstra’s Algorithm.

This project demonstrates real-world application of **Data Structures, Algorithms, Computer Organization & Architecture concepts, and Software Engineering principles**.

---

## ⚙️ Features
- Add bus stops dynamically
- Create routes between stops with distances
- View all available bus stops and routes
- Find shortest path between two stops
- Menu-driven console interface

---

## 🧠 Technologies Used
- Java (Core)
- Data Structures (Graph, HashMap, PriorityQueue)
- Algorithms (Dijkstra’s Shortest Path)
- Object-Oriented Programming

---

## 🚀 How to Run

```bash
javac *.java
java Main
```

---

## 📊 Algorithm Used

- Dijkstra’s Algorithm for shortest path
- PriorityQueue for optimization
- Adjacency List for graph representation
